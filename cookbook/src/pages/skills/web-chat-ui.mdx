---
title: Customize your chat UI
description: How to customize your chat UI in IBM Watson™ Assistant
---

When you use the sample skill provided, you need to customize your chat UI to specify the  three conversation starters text. The conversation starter text is preconfigured as part of the welcome message in the sample skill.

The following list outlines how to update the home screen text:

1. From your assistant, under the Integrations section, select `web chat`.
2. From the `Style` tab, update `Virtual Assistant` as the assistant's name as known by customers.
3. From the `Home screen` tab, update the following and click `Save and exit`.

The following table outlines the three conversation starters text to specify.

| Field Name | Value |
| :------ | :------ |
| Greeting message | Hi! I’m your virtual assistant. Let me help you with your caseworker tasks.|
| Conversation starter 1 | Outstanding verifications by case |
| Conversation starter 2 | Outstanding verifications by client |
| Conversation starter 3 | Caseworker procedures |
