## IBM Cloud Services

You will need:

+ An IBM Cloud account.
+ An instance of an IBM Watson Assistant service.

All of the above may be done through the [IBM Cloud site](https://cloud.ibm.com/).

### Decide how you want to set up your IBM Watson Assistant instance

The following list outlines two scenarios that you can use:

+ Use an IBM Watson Assistant instance on the IBM public cloud.
+ Deploy IBM Watson Assistant on your private cloud or data center by using IBM Red Hat OpenShift and IBM Cloud Pak for Data.

See the [Deployment options](/architecture/deployment-options) for more information.
