title: Set up IBM Watson Assistant
description: Options for how to set up IBM Watson Assistant
---

Decide how you want to set up your IBM Watson Assistant instance. The following list outlines two scenarios that you can use:

+ Use an IBM Watson Assistant instance on the IBM public cloud.
+ Deploy IBM Watson Assistant on your private cloud or data center by using IBM Red Hat OpenShift and IBM Cloud Pak for Data.

Set up your IBM Watson Assistant and you are ready to start working.
